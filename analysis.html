Analysis

In this section we are going to do the following three analysis: Correlation Matrix, Wilcoxon Rank-sum Test and Regression.

1. Wilcoxon Rank-sum Test

We are going to Compare the NY State residents and nursing home seniors’ medians level of fatality rate and infection rate.

a) Compare the NY State residents and nursing home seniors’ medians level of fatality rate.

test normality

nursing_house_df =

hist(residents_df$monthly_fatality_rate, xlab = "monthly_fatality_rate", freq=T, main="residents monthly fatality rate")

qqnorm(nursing_house_df$nh_monthly_fatality_rate, col=2, pch=19, cex=1.5)
qqline(nursing_house_df$nh_monthly_fatality_rate, col = 1,lwd=2,lty=2)

qqline(residents_df$monthly_fatality_rate, col = 1,lwd=2,lty=2)

From the Histogram and QQ Plot, we can see that the distribution of nursing home monthly fatality rate and residents monthly fatality rates are heavily right skewed. Therefore, they violated the normality assumptions. We are going to use non-parametric test: Wilcoxon Rank - Sum Test to do the following hypothesis test:

H0: The medians level of monthly fatality rate are the same for NY State residents and nursing home seniors.

Ha: The medians level of monthly fatality rate are different for NY State residents and nursing home seniors.

res <- wilcox.test(nursing_house_df$nh_monthly_fatality_rate, residents_df$monthly_fatality_rate, mu=0)
nursing_house_df %>% count()

## 1   172
residents_df %>% count()
## # A tibble: 1 x 1
##       n
##   <int>
## 1   300
res$statistic<-res$statistic + 300*(300+1)/2
res
## 
##  Wilcoxon rank sum test with continuity correction
## 
## data:  nursing_house_df$nh_monthly_fatality_rate and residents_df$monthly_fatality_rate
## W = 72365, p-value = 0.3194
## alternative hypothesis: true location shift is not equal to 0

From the Wilcoxon rank sum test with continuity correction, we can see the p-value is 0.3194 > 0.05. Therefore, we fail to reject null hypothesis. We are 95% confident that the medians level of monthly fatality rate are the same for NY State residents and nursing home seniors.

# format the correlation matrix
flattenCorrMatrix <- function(cormat, pmat) {
  ut <- upper.tri(cormat)
  data.frame(
    row = rownames(cormat)[row(cormat)[ut]],
    column = rownames(cormat)[col(cormat)[ut]],
    cor  =(cormat)[ut],
    p = pmat[ut]
    )
}

3. Multiple Linear Regression with Panel Data

The Data is containing information with county and date. To better eliminate bias and capture entity effect and time effect in the model, we first try a panel data regression.

Note here we are exploring the nursing home prevalence rate as dependent variable.

a)MLR model: panel data regression using backward elimination to select features

  1. filter NAs in predictors
  2. Select potential predictors according to correlation matrix and epidemiology concept
  3. create initial model with longitudinal data
  4. using backward elimination to do feature selection
  5. diagnostics with homoskedasticity and residual
my_data_lm =
read_csv("./data/final_df.csv") %>%
  janitor::clean_names() %>% 
  relocate(election_result) %>% 
  mutate(
   election_result = ifelse(election_result=="D", "1","0")
   ) %>% 
  mutate(
    election_result = as.numeric(election_result),
    county = as.factor(county),
    month = as.factor(month)
  ) %>% 
  mutate(
      nh_occupancy_rate = ifelse(nh_occupancy_rate >= 1, NA,nh_occupancy_rate)
  ) %>% 
  drop_na() %>% 
  filter_all(all_vars(is.finite(.)))
# all potential predictors
multi_fit =
  lm(county_monthly_prevalence
     ~ election_result
     + nh_occupancy_rate
     + staff_flu_vaccination_rate
     + total_shortage_stuff_monthly
     + ventilator_dependent_unit_monthly
     + monthly_infection_rate
     + monthly_fatality_rate,
     
     data = my_data
  )
summary(multi_fit)

The result table shows a model summary for several potential predictors to best predict nursing home monthly prevalence rate. From the best fitted regression model(1), we learned that

  1. out of expectation, we conclude that election result cannot affect nursing home COVID prevalence. This is probably affected by the fact that we are using binary election outcome, if we use a continuous measurement of election and political belief, it might change the regression result.

  2. ventilator can decrease nursing home COVID prevalence, although in a small amount. If there is sufficient ventilator in the facility, COVID prevalence wold decrease by 2%

  3. the nursing home COVID prevalence is largely dependent on infection rate of its county. Each 0.1% increase in the county infection rate would increase nursing home prevalence by 38.3%

  4. each 10% increase in county’s fatality rate would increase 130% nursing home prevalence

4. MLR Without panel data form

We continue to use simple multiple linear regression to predict the nursing home prevalence. This will give us a more generalized model.

We use AIC regression for the modeling.

a) MLR stepwise AIC regression

  1. convert data to non-longitudinal form
  2. use step AIC regression to perform feature selection
  3. diagnostics
# clear entity and states
no.na.data = 
  my_data_lm %>% 
  na.omit() %>% 
  dplyr::select(
    -county,
    -month

step_ml = step(aic_res, direction='backward', trace = 0)
